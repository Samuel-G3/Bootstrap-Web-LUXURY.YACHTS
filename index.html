Admiral C Force 65

Aggressive and modern lines: these are the elements that characterize the series C FORCE. Among the most significant details of this class is certainly the large central double height glazing that meets the need to flow with as much natural light as possible to where it is needed most: the salon on the main and upper decks. In this way it is created the typical atmosphere of an attic. Seen from outside, the window appears as a vertical cut that does not interrupt in any way the external lines. All these features make it special series C FORCE, but the proportions are well balanced to make it timeless.
